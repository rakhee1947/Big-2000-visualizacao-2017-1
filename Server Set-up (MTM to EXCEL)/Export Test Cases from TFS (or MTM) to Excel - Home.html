    This tool allows the user to export test cases from TFS (or MTM) to Microsoft Excel.
    It is really easy to use, user just need to select a test suite and specify a destination folder and file name.

    The tool provides certain options to the user to specify how the export should work if there are sub-suites in the selected suite.

    The user can also specify if they want the test case results to be exported.

    This tool works with Team foundation server 2012 as well as with the 2010 version.
